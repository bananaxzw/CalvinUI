Drag(拖拉)

依赖
脚本文件:jquery.js
参数
  • options
    Object
    可选

    多选形式的可配置参数。

以下是option参数的说明

参数 数据类型 默认值 说明
containment HTMLDomElement null

拖拉的限定容器范围
axis string('v'|'h') null

拖拉的限定方向
handle string selector|HTMLDOMEelement

拖拉的手柄

实例1

基本用法

<script type="text/javascript">
$("#drag_basic").CalvinDraggable();
 </script>

实例1

限定容器

<script type="text/javascript">
  $("#drag_container").CalvinDraggable({ containment: document.getElementById("dragContainer") });
 </script>
